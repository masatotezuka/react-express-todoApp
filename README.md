# react-express-todoApp

## 環境構築
#### 1. レポジトリをローカルにクローンする.
```
git clone https://github.com/masatotezuka/react-express-todoApp.git
```

#### 2. npmのインストール
```
npm i
```

#### 3.DBのマイグレーション
```
npx sequelize-cli db:migrate
```

#### 4. バックエンドのサーバーを立ち上げる
```
npm start
```

#### 5. フロントエンドのサーバーを立ち上げる
```
cd client
npm start
```
